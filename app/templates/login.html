Hello and welcome to Quiz Manager!

Quiz Manager is a web application for saving quizzes and lets users view them through an online portal.

There are 3 available users, each with a different level of permission:

  • restrictedUser: view quizzes and questions, but not answers

  • user: view all quizzes, questions and answers

  • admin: same as user but also has permission to add, modify and delete data

The password for all users is "password".

Any changes made will be persisted in the database. The database will be reset back to its original state every 24 hours.

For more information please see the GitHub repository.
